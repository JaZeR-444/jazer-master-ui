Key Features

Dual Layer Glow

Two overlapping layers create depth: a blurred gradient background and a solid overlay.

Neon Text Shadow

Multi-layered text shadow creates authentic neon sign glow effect on hover.

Gradient Background

Blurred gradient (cyan → pink → purple) provides vibrant cyberpunk aesthetics.

Smooth Animation

All effects transition smoothly using 0.4s easing for polished interactions.

Hover Lift

Links lift upward on hover for enhanced feedback and depth perception.

Rounded Design

Soft border radius creates modern, friendly appearance matching current design trends.

Usage Notes
