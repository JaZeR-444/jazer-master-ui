Breadcrumb Trail Fade-In

Breadcrumb navigation with sequential fade-in animation. Perfect for showing hierarchical page structure with style.

Home ›
Products ›
Electronics ›
Laptops ›
Gaming Laptops

HTML Structure

<nav class="breadcrumb" aria-label="Breadcrumb">
    <div class="breadcrumb__item">
        <a href="#" class="breadcrumb__link">Home</a>
        <span class="breadcrumb__separator">›</span>
    </div>
    <div class="breadcrumb__item">
        <a href="#" class="breadcrumb__link">Products</a>
        <span class="breadcrumb__separator">›</span>
    </div>
    <div class="breadcrumb__item">
        <span class="breadcrumb__current">Current Page</span>
    </div>
</nav>

CSS (Key Styles)

.breadcrumb {
    display: flex;
    align-items: center;
    gap: 0.75rem;
}

.breadcrumb__item {
    opacity: 0;
    animation: fadeInBreadcrumb 0.5s ease forwards;
}

/* Staggered delays */
.breadcrumb__item:nth-child(1) { animation-delay: 0s; }
.breadcrumb__item:nth-child(2) { animation-delay: 0.15s; }
.breadcrumb__item:nth-child(3) { animation-delay: 0.3s; }

@keyframes fadeInBreadcrumb {
    from {
        opacity: 0;
        transform: translateX(-10px);
    }
    to {
        opacity: 1;
        transform: translateX(0);
    }
}
