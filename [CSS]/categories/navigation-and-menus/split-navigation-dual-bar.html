Split Navigation Dual Bar

Separate top and bottom navigation bars for different content categories

Overview

This pattern splits navigation into two bars: a top bar for primary site navigation and a bottom bar for secondary actions or context-specific controls.

The dual-bar approach maximizes usability by organizing navigation hierarchically while keeping frequently-used actions accessible at all times.

Key Features

Hierarchical Structure

Top bar for main navigation, bottom bar for actions

Fixed Positioning

Both bars stay visible during scrolling

Visual Distinction

Different colors separate top and bottom contexts

Mobile Optimized

Icon-based bottom nav perfect for mobile apps
