Key Features

Animated Beam

The beam indicator animates smoothly from 0 to full width when a route becomes active.

Pulsing Glow

Active route features a continuous pulsing glow effect for enhanced visibility.

Hover Effects

Links glow with cyan color and text shadow on hover for interactive feedback.

Gradient Beam

Uses JaZeR's signature gradient (cyan to pink to purple) for the beam indicator.

Smooth Transitions

Cubic-bezier easing creates natural, bouncy animations for better UX.

Accessible

Includes ARIA labels and proper semantic HTML for screen readers.

Usage Notes

  • Click any navigation link to activate the beam indicator
  • Hover over links to see the glow and text shadow effects
  • The active beam continuously pulses for visual emphasis
  • Perfect for dashboard navigation, application tabs, or website headers
  • Fully responsive with touch-friendly spacing on mobile
  • Uses CSS custom properties from JaZeR brand design system
