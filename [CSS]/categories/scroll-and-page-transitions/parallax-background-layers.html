Layer Controls

Parallax Background Layers

Multiple background layers that move at different speeds as you scroll, creating immersive depth and dimension in your designs.

Layered Depth

Each background layer moves at its own unique speed, creating a sense of three-dimensional space. Distant elements move slowly while closer elements move faster, mimicking real-world parallax.

This technique adds visual interest and draws users deeper into your content.

Five Distinct Layers

Stars: The deepest layer, moving at 10% scroll speed, creating a distant cosmic backdrop.

Nebulae: Colorful cloud formations at 20% speed add atmospheric depth.

Shapes: Geometric elements at 35% create mid-ground visual interest.

Grid & Particles

Grid: A subtle overlay at 50% speed provides structure and cyberpunk aesthetic.

Particles: The fastest layer at 70% adds dynamic sparkle and energy to the foreground.

Together, these layers create an immersive scrolling experience.

Performance Optimized

GPU-accelerated CSS transforms ensure silky smooth 60fps animations. The will-change property hints to the browser for optimal rendering.

RequestAnimationFrame throttling prevents unnecessary calculations during rapid scrolling.

Fully Customizable

Use the control panel to switch between visual styles, adjust parallax speed, and change movement direction.

Perfect for hero sections, landing pages, portfolios, and any site where you want to make a lasting impression.
