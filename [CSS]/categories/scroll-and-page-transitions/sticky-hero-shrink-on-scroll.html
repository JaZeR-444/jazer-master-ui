Smooth Shrinking Animation

The hero section uses position: sticky to remain at the top of the viewport as you scroll. When you start scrolling, it smoothly shrinks from a full-height hero into a compact navigation header, creating a seamless transition that maintains context while maximizing content space.

This effect is perfect for landing pages, portfolios, and any site where you want to make a bold first impression that elegantly transforms into a persistent navigation element. The transition is powered by CSS transforms and is hardware-accelerated for smooth 60fps performance.

Interactive Experience

Scroll up and down to see the hero section expand and shrink. Notice how the transition is smooth and reversible - the hero grows back to full size when you scroll to the top, and shrinks again as you scroll down. The compact header reveals navigation links while hiding less critical information.

The effect uses JavaScript to detect scroll position and add a "shrunk" class when scrolling down. All the heavy lifting is done by CSS transitions, ensuring optimal performance across all devices and browsers. This approach provides the best of both worlds: the flexibility of JavaScript detection with the performance of CSS animations.

Implementation Details

The sticky hero shrink effect combines several modern CSS features: position sticky for the scroll behavior, CSS transforms for smooth scaling, and CSS transitions for timing control. The layout uses flexbox for easy centering and alignment, while maintaining accessibility with proper semantic HTML structure.

Performance is optimized by using transform properties (scale, translate) which are hardware-accelerated, rather than animating layout properties like height or padding directly. This ensures the animation remains smooth even on lower-end devices. The transition timing uses cubic-bezier easing for a natural, professional feel.
