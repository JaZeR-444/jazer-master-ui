Pressed Depth Button

A button with realistic depth effect that lifts on hover and presses down when clicked, creating tactile visual feedback.

CSS Code

.pressed-depth {
    background: linear-gradient(135deg, var(--jazer-blue-mid), var(--jazer-purple));
    border: none;
    color: var(--text-light);
    padding: 1rem 2rem;
    font-family: var(--font-secondary);
    font-weight: 700;
    text-transform: uppercase;
    letter-spacing: 0.05em;
    border-radius: 8px;
    position: relative;
    transition: all 0.1s ease;
    box-shadow: 0 6px 0 rgba(147, 51, 234, 0.3), 0 8px 10px rgba(0, 0, 0, 0.3);
    cursor: pointer;
}

.pressed-depth:hover {
    transform: translateY(-2px);
    box-shadow: 0 8px 0 rgba(147, 51, 234, 0.3), 0 10px 15px rgba(0, 0, 0, 0.3);
}

.pressed-depth:active {
    transform: translateY(4px);
    box-shadow: 0 2px 0 rgba(147, 51, 234, 0.3), 0 4px 5px rgba(0, 0, 0, 0.3);
}
