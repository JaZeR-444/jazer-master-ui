Ghost Neon Button

A transparent button with neon glow text that fills with solid color on hover, creating a ghost-to-solid transformation effect.

CSS Code

.ghost-neon {
    background: transparent;
    border: 2px solid var(--jazer-cyan);
    color: var(--jazer-cyan);
    padding: 1rem 2rem;
    font-family: var(--font-secondary);
    font-weight: 700;
    text-transform: uppercase;
    letter-spacing: 0.05em;
    border-radius: 8px;
    position: relative;
    transition: all 0.3s ease;
    text-shadow: 0 0 10px rgba(0, 242, 234, 0.5);
    cursor: pointer;
}

.ghost-neon:hover {
    background: var(--jazer-cyan);
    color: var(--bg-dark);
    text-shadow: none;
    box-shadow: 0 0 30px rgba(0, 242, 234, 0.8), inset 0 0 20px rgba(255, 255, 255, 0.2);
    transform: translateY(-2px);
}
