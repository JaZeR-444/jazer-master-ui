Icon-Only Pulse Button

A circular icon button with expanding pulse rings on hover, perfect for floating action buttons and icon-based interactions.

CSS Code

.icon-pulse {
    background: var(--gradient-accent);
    border: none;
    color: var(--text-light);
    width: 60px;
    height: 60px;
    border-radius: 50%;
    display: flex;
    align-items: center;
    justify-content: center;
    font-size: 1.5rem;
    position: relative;
    transition: all 0.3s ease;
    cursor: pointer;
}

.icon-pulse::before {
    content: '';
    position: absolute;
    top: -10px;
    left: -10px;
    right: -10px;
    bottom: -10px;
    border: 2px solid var(--jazer-pink);
    border-radius: 50%;
    opacity: 0;
    animation: iconPulse 2s ease-in-out infinite;
}

.icon-pulse:hover::before {
    opacity: 1;
}

.icon-pulse:hover {
    transform: scale(1.1);
    box-shadow: 0 0 20px rgba(255, 0, 110, 0.5);
}

@keyframes iconPulse {
    0% { transform: scale(1); opacity: 1; }
    100% { transform: scale(1.3); opacity: 0; }
}
