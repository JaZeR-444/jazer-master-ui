Circular Progress Button

A circular button with rotating gradient border that creates a loading/progress effect, perfect for action buttons and loading states.

CSS Code

.circular-progress {
    position: relative;
    width: 80px;
    height: 80px;
    border-radius: 50%;
    background: var(--gradient-accent);
    border: none;
    color: var(--text-light);
    font-family: var(--font-secondary);
    font-weight: 700;
    cursor: pointer;
    overflow: hidden;
    transition: all 0.3s ease;
}

.circular-progress::before {
    content: '';
    position: absolute;
    top: -2px;
    left: -2px;
    right: -2px;
    bottom: -2px;
    border-radius: 50%;
    background: conic-gradient(from 0deg, var(--jazer-cyan), var(--jazer-purple), var(--jazer-pink), var(--jazer-cyan));
    animation: circularRotate 2s linear infinite;
    z-index: -1;
}

.circular-progress:hover {
    transform: scale(1.1);
    box-shadow: 0 0 30px rgba(255, 0, 110, 0.5);
}

@keyframes circularRotate {
    0% { transform: rotate(0deg); }
    100% { transform: rotate(360deg); }
}
