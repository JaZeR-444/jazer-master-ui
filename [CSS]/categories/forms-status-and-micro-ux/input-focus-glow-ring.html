Input Focus Glow Ring

Input field with glow ring on focus.

Code

<input type="text" class="glow-input" placeholder="Focus me">

.glow-input {
    width: 100%;
    padding: 12px 16px;
    background: rgba(255, 255, 255, 0.05);
    border: 2px solid rgba(255, 255, 255, 0.1);
    border-radius: 8px;
    color: white;
    font-size: 16px;
    transition: all 0.3s ease;
}

.glow-input:focus {
    outline: none;
    border-color: #00f2ea;
    box-shadow: 0 0 0 3px rgba(0, 242, 234, 0.3);
}
