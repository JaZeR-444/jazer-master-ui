Responsive Modal

This modal demonstrates responsive behavior across different screen sizes. On larger screens, it maintains a standard desktop layout with side-by-side buttons.

On tablets and mobile devices, the modal adjusts its size, layout, and interaction patterns to provide an optimal viewing experience.

Key responsive features:

  • Adjusts width and height based on screen size
  • Changes button layout from horizontal to vertical on mobile
  • Adapts header layout for smaller screens
  • Maintains accessibility standards across devices
  • Optimizes touch targets for mobile interaction
