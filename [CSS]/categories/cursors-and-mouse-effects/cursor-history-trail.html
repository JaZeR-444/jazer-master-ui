Features

  • Position History: Stores last 20 cursor positions
  • Fade Effect: Older dots fade out progressively
  • Size Variation: Dots get smaller towards the tail
  • Smooth Animation: 60fps rendering with RAF
  • Dynamic Trail: Adapts to cursor speed
  • Low Overhead: Efficient DOM reuse

Use Cases

  • Gaming UIs: Show movement history in strategy games
  • Drawing Apps: Preview brush or pen trail
  • Data Viz: Track user attention on charts
  • Presentations: Highlight cursor movement for audience
  • Tutorials: Show user where to move mouse
  • Creative Sites: Artistic trailing effect
