Snake/Worm Cursor

Playful snake cursor with segments that follow your cursor path

Move Around

Watch the snake segments follow your cursor path with smooth trailing

Features

  • Multi-Segment Trail: 15 connected segments follow cursor path
  • Smooth Following: Each segment follows previous with delay
  • Gradient Colors: Cyan to purple gradient across segments
  • Glow Effect: Neon glow around each segment
  • Performance: Optimized position tracking at 60fps
  • Responsive: Adapts to all screen sizes

Use Cases

  • Gaming Interfaces: Retro snake game aesthetics
  • Kids Websites: Playful, engaging cursor for children
  • Creative Portfolios: Unique artistic expression
  • Interactive Art: Drawing and painting applications
  • Event Pages: Fun cursor for festival/party websites
  • Educational: Teach movement and trailing concepts

Customization Options

Snake Properties

  • Adjust number of segments (5-50)
  • Change segment size (10px-40px)
  • Modify following distance (spacing between segments)
  • Set follow speed (how quickly segments catch up)

Visual Styling

  • Custom color schemes (solid, gradient, rainbow)
  • Vary segment shapes (circle, square, custom)
  • Add animation (pulse, rotate, scale)
  • Enable fade-out on tail segments

Implementation

const segmentCount = 15;
const segments = [];

// Create segments
for (let i = 0; i < segmentCount; i++) {
  const segment = document.createElement('div');
  segment.className = 'snake-segment';
  segment.style.opacity = 1 - (i / segmentCount) * 0.5;
  document.body.appendChild(segment);
  segments.push({ el: segment, x: 0, y: 0 });
}

let mouseX = 0, mouseY = 0;

document.addEventListener('mousemove', (e) => {
  mouseX = e.clientX;
  mouseY = e.clientY;
});

function animate() {
  // Head follows mouse
  segments[0].x += (mouseX - segments[0].x) * 0.3;
  segments[0].y += (mouseY - segments[0].y) * 0.3;
  
  // Other segments follow previous
  for (let i = 1; i < segmentCount; i++) {
    segments[i].x += (segments[i-1].x - segments[i].x) * 0.3;
    segments[i].y += (segments[i-1].y - segments[i].y) * 0.3;
  }
  
  // Update positions
  segments.forEach((seg, i) => {
    seg.el.style.transform = `translate(${seg.x - 10}px, ${seg.y - 10}px)`;
  });
  
  requestAnimationFrame(animate);
}
animate();
