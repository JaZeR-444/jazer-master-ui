Features

  • Vector-Based Rotation: Arrow calculates angle from movement velocity
  • Smooth Transitions: CSS transforms with easing for fluid rotation
  • High Performance: Uses requestAnimationFrame for 60fps updates
  • Visual Feedback: Cyan arrow with purple stroke and glow effect
  • Responsive Design: Adapts to all screen sizes and devices
  • Zero Lag: Efficient calculation using Math.atan2 for instant response

Use Cases

  • Navigation Interfaces: Show direction of available movement paths
  • Gaming UIs: Indicate character or camera rotation direction
  • Data Visualization: Display trend direction on hover over graphs
  • Map Applications: Show travel direction or heading
  • Drawing Tools: Indicate brush direction or stroke angle
  • Educational Content: Teach vector mathematics and physics concepts

Customization Options

Arrow Design

  • Change arrow shape (triangle, chevron, custom SVG path)
  • Adjust size (default 40px, range 20-80px recommended)
  • Modify colors (fill, stroke, glow effects)
  • Add additional indicators (speed bars, trail effect)

Rotation Behavior

  • Adjust rotation speed/smoothing (transition duration)
  • Set minimum movement threshold to trigger rotation
  • Add snap angles (45° increments for 8-directional)
  • Enable/disable rotation when stationary

Visual Enhancements

  • Add speed-based effects (faster = longer trail)
  • Implement directional color shifts
  • Add particle trails in movement direction
  • Include velocity indicator (arrow length scales with speed)

Performance Notes

  • Optimized Calculation: Math.atan2 is hardware-accelerated on modern CPUs
  • GPU Acceleration: CSS transforms use hardware acceleration
  • Minimal Repaints: Only cursor element updates, no layout thrashing
  • Smooth at 60fps: requestAnimationFrame ensures optimal frame timing
  • Low Memory: Only stores previous position (8 bytes)
  • Battery Friendly: Efficient calculations don't tax mobile devices

Implementation

<!-- HTML Structure -->
<div id="cursor-arrow">
  <svg viewBox="0 0 40 40" fill="none">
    <path d="M20 5 L35 20 L20 35 L20 25 L5 25 L5 15 L20 15 Z" 
          fill="#00f2ea" stroke="#9333ea" stroke-width="2"/>
  </svg>
</div>

<style>
  body {
    cursor: none; /* Hide default cursor */
  }

  #cursor-arrow {
    position: fixed;
    width: 40px;
    height: 40px;
    pointer-events: none;
    z-index: 9999;
    transition: transform 0.15s ease-out;
    will-change: transform;
  }

  #cursor-arrow svg {
    filter: drop-shadow(0 0 10px #00f2ea);
  }
</style>

<script>
  const cursorArrow = document.getElementById('cursor-arrow');
  let mouseX = 0, mouseY = 0;
  let prevX = 0, prevY = 0;
  let currentAngle = 0;

  // Track mouse movement
  document.addEventListener('mousemove', (e) => {
    mouseX = e.clientX;
    mouseY = e.clientY;
  });

  // Update cursor position and rotation
  function updateCursor() {
    // Calculate velocity
    const deltaX = mouseX - prevX;
    const deltaY = mouseY - prevY;

    // Calculate angle from velocity (if moving)
    if (Math.abs(deltaX) > 0.5 || Math.abs(deltaY) > 0.5) {
      currentAngle = Math.atan2(deltaY, deltaX) * (180 / Math.PI);
    }

    // Update position and rotation
    cursorArrow.style.transform = 
      `translate(${mouseX - 20}px, ${mouseY - 20}px) rotate(${currentAngle}deg)`;

    // Store current position
    prevX = mouseX;
    prevY = mouseY;

    requestAnimationFrame(updateCursor);
  }

  // Start animation loop
  updateCursor();
</script>
