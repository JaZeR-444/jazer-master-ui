Like / Favorite Button

Icon that toggles filled/outline.

Code

<button class="like-button liked">
    <span class="like-icon">❤️</span>
    <span class="like-count">43</span>
</button>

.like-button {
    background: none;
    border: 1px solid #dee2e6;
    border-radius: 20px;
    padding: 8px 16px;
    font-size: 14px;
    cursor: pointer;
    transition: all 0.2s;
    display: flex;
    align-items: center;
    gap: 6px;
    color: #6c757d;
}

.like-button:hover {
    background: #f8f9fa;
    border-color: #adb5bd;
}

.like-button.liked {
    background: #dc3545;
    border-color: #dc3545;
    color: white;
}

.like-button.liked:hover {
    background: #c82333;
    border-color: #c82333;
}

.like-icon {
    font-size: 16px;
    transition: transform 0.2s;
}

.like-button.liked .like-icon {
    transform: scale(1.1);
}
