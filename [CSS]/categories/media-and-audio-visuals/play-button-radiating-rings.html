Play Button Radiating Rings

Play button with radiating ring animations.

Code

<button class="play-button">▶</button>

.play-button {
    width: 80px;
    height: 80px;
    background: linear-gradient(135deg, #00f2ea 0%, #ff006e 50%, #9333ea 100%);
    border: none;
    border-radius: 50%;
    cursor: pointer;
    position: relative;
    display: flex;
    align-items: center;
    justify-content: center;
    font-size: 1.5rem;
    color: white;
    transition: all 0.3s ease;
}

.play-button::before {
    content: '';
    position: absolute;
    top: -10px;
    left: -10px;
    right: -10px;
    bottom: -10px;
    border: 2px solid #00f2ea;
    border-radius: 50%;
    animation: radiate 2s ease-in-out infinite;
}

.play-button::after {
    content: '';
    position: absolute;
    top: -20px;
    left: -20px;
    right: -20px;
    bottom: -20px;
    border: 1px solid #ff006e;
    border-radius: 50%;
    animation: radiate 2s ease-in-out infinite reverse;
}

.play-button:hover {
    transform: scale(1.1);
}

@keyframes radiate {
    0%, 100% { transform: scale(1); opacity: 0.5; }
    50% { transform: scale(1.2); opacity: 0; }
}
