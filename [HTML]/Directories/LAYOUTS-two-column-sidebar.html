Two-Column Layout with Sticky Sidebar

This layout pattern is perfect for documentation sites, blogs with sidebars, dashboards, and any content-heavy page that benefits from persistent navigation.

Key Features

  • Sticky Sidebar: Sidebar stays visible as you scroll down the page
  • Responsive: Stacks to single column on mobile devices
  • Grid Layout: Uses CSS Grid for clean, flexible structure
  • Sidebar First on Mobile: Content appears before sidebar on small screens (change order with CSS order property)

How It Works

The layout uses CSS Grid with two columns: a fixed-width sidebar (300px) and a flexible main content area (1fr). The sidebar has position: sticky which keeps it in view as you scroll.

On screens smaller than 768px, the grid switches to a single column and the sidebar loses its sticky positioning for better mobile UX.

Customization Options

You can easily customize this layout:

  • Sidebar width: Change grid-template-columns: 300px 1fr to your preferred width
  • Sidebar position: Swap the grid template to 1fr 300px to move sidebar to the right
  • Gap size: Adjust the gap: 2rem property
  • Sticky offset: Change top: 2rem to adjust where sidebar sticks
  • Mobile order: Use order property to change stacking order on mobile

Use Cases

  • Documentation websites (like this example)
  • Blog posts with table of contents
  • E-commerce product pages with filters
  • Dashboard layouts
  • Settings pages
  • Course or tutorial platforms
