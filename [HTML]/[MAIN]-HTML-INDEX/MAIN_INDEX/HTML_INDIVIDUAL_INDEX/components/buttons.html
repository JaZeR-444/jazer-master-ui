Usage Notes

  • Combine classes for different variations (e.g., btn btn--primary btn--large)
  • Use :disabled or btn--disabled class for disabled state
  • Icons work best with inline SVGs for color control
  • Loading state automatically centers spinner and hides text
  • Always include focus states for accessibility
  • Minimum touch target: 44x44px (especially for mobile)
