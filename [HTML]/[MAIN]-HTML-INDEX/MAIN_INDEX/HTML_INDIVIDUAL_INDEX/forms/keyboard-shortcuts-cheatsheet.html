Keyboard Shortcuts Cheatsheet

Boost your productivity with these useful keyboard shortcuts.

General Actions

Save Document
Ctrl + S
Undo Last Action
Ctrl + Z
Redo Action
Ctrl + Y
Print Page
Ctrl + P
Find in Page
Ctrl + F
Open Settings
Ctrl + ,

Navigation

Next Tab
Ctrl + Tab
Previous Tab
Ctrl + Shift + Tab
Go Back
Alt + Left Arrow
Go Forward
Alt + Right Arrow

Editor Actions (Example)

Copy Line Up
Shift Alt + Up Arrow
Delete Line
Ctrl Shift + K
