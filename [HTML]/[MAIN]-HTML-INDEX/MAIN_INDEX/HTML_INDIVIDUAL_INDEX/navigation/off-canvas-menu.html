Off-Canvas Menu Examples

Slide-out navigation panels from different directions

How to Use

Click the buttons in the header to open different off-canvas menu variations. Each menu demonstrates a different approach to slide-out navigation.

Left Menu: Slides in from the left with an overlay
Right Menu: Slides in from the right with user profile
Full Menu: Full-screen overlay navigation
Push Menu: Pushes the content to the right

Features

All menus include smooth animations, accessibility features, and responsive design. They work on all modern browsers and mobile devices.
