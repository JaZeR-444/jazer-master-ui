Best Practices:

  • Use <nav> with aria-label="Breadcrumb" for accessibility
  • Use an ordered list (<ol>) since breadcrumbs represent a hierarchy
  • Mark the current page with aria-current="page"
  • Hide separators from screen readers using aria-hidden="true"
  • Keep breadcrumb text concise (truncate long titles if needed)
  • Don't include the current page as a link
