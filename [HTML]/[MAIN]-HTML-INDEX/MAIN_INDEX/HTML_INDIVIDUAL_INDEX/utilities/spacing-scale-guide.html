Spacing Scale Guide

A visual guide to the consistent spacing tokens used across the application, ensuring harmonious layouts and predictable rhythm.

Visual Spacing Tokens

--space-xxs (4px)

Extra extra small spacing.

--space-xs (8px)

Extra small spacing.

--space-sm (12px)

Small spacing.

--space-md (16px)

Medium spacing. (Default base unit)

--space-lg (24px)

Large spacing.

--space-xl (32px)

Extra large spacing.

--space-xxl (48px)

Extra extra large spacing.

--space-xxxl (64px)

Extra extra extra large spacing.
