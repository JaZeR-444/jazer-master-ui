Three-Column Layout Pattern

This layout features a left sidebar for navigation, a main content area, and a right sidebar for supplementary information. It's commonly used for documentation sites, news websites, and blogs.

Grid Configuration

The layout uses CSS Grid with this configuration:

  • Left Sidebar: 250px fixed width
  • Main Content: 1fr (flexible, takes remaining space)
  • Right Sidebar: 300px fixed width
  • Gap: 2rem between columns

Responsive Behavior

The layout adapts intelligently across different screen sizes:

  • Desktop (>1024px): All three columns visible
  • Tablet (768px - 1024px): Left sidebar + main content (right sidebar hidden)
  • Mobile (<768px): Single column, stacked vertically

Sticky Sidebars

Both sidebars use position: sticky to remain visible while scrolling through long content. This makes navigation and supplementary information always accessible.

Use Cases

  • Documentation websites with table of contents and related links
  • News sites with category nav and trending stories
  • Blogs with categories and recent posts/popular tags
  • E-commerce product listings with filters and promotions
  • Dashboards with navigation and quick stats

Customization Tips

Adjust the layout to your needs:

  • Change column widths: Modify grid-template-columns: 250px 1fr 300px
  • Swap sidebar positions: Change the order in HTML or use CSS Grid's order property
  • Remove a sidebar: Simply delete it from HTML or set display: none
  • Adjust breakpoints: Modify the media query values (1024px, 768px) to suit your design
  • Full-width main: Set sidebars to display: none on certain pages
