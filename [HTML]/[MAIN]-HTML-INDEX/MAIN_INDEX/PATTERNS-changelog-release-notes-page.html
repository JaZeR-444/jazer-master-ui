Changelog & Release Notes

Version 2.1.0

December 2, 2023

New Features

  • Introduced dark mode toggle for personalized user experience.
  • Added real-time collaboration features to document editor.
  • New dashboard widgets for advanced analytics overview.

Improvements

  • Enhanced performance of data tables with virtualization.
  • Improved mobile responsiveness across all core pages.

  • Fixed an issue where some users could not reset their password.
