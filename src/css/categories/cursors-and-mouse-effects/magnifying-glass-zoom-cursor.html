Magnifying Glass Zoom Inside Cursor

Interactive magnifying glass effect that zooms content under the cursor

Hover to Magnify

Move your cursor over this area to see the magnification effect. The magnifying glass reveals zoomed-in details of the content beneath it.

Features

  • Real-Time Zooming: Canvas-based magnification at 2x scale
  • Smooth Tracking: Magnifier follows cursor with no lag
  • Visual Feedback: Cyan border with glow and crosshairs
  • Area Detection: Only appears over demo content
  • High Performance: GPU-accelerated canvas rendering
  • Responsive: Adjusts size on mobile devices

Use Cases

  • E-commerce: Product detail inspection for jewelry, fabric textures
  • Medical Imaging: Detailed examination of X-rays and scans
  • Maps: Street-level detail inspection on large map views
  • Photo Galleries: Fine detail viewing without full zoom
  • Technical Drawings: CAD, blueprints, and schematic inspection
  • Accessibility: Assistive tool for users with visual impairments

Implementation

<!-- HTML Structure -->
<div id="magnifier">
  <canvas id="magnifier-canvas"></canvas>
</div>

<div id="demo-area">
  <!-- Content to magnify -->
</div>

<style>
  #demo-area {
    cursor: none;
  }

  #magnifier {
    position: fixed;
    width: 150px;
    height: 150px;
    border: 4px solid #00f2ea;
    border-radius: 50%;
    pointer-events: none;
    z-index: 9999;
    box-shadow: 0 0 20px rgba(0, 242, 234, 0.6);
    opacity: 0;
    overflow: hidden;
  }
</style>

<script>
  const magnifier = document.getElementById('magnifier');
  const canvas = document.getElementById('magnifier-canvas');
  const ctx = canvas.getContext('2d');
  const demoArea = document.getElementById('demo-area');
  
  const size = 150;
  const zoom = 2;
  
  canvas.width = size;
  canvas.height = size;

  let isOverDemo = false;

  demoArea.addEventListener('mouseenter', () => {
    isOverDemo = true;
    magnifier.style.opacity = '1';
  });

  demoArea.addEventListener('mouseleave', () => {
    isOverDemo = false;
    magnifier.style.opacity = '0';
  });

  document.addEventListener('mousemove', (e) => {
    if (!isOverDemo) return;

    const x = e.clientX;
    const y = e.clientY;

    // Position magnifier
    magnifier.style.left = (x - size / 2) + 'px';
    magnifier.style.top = (y - size / 2) + 'px';

    // Draw magnified content
    ctx.clearRect(0, 0, size, size);
    ctx.save();
    
    // Create circular clipping path
    ctx.beginPath();
    ctx.arc(size / 2, size / 2, size / 2, 0, Math.PI * 2);
    ctx.clip();
    
    // Draw magnified portion
    const sourceX = x - (size / zoom / 2);
    const sourceY = y - (size / zoom / 2);
    const sourceSize = size / zoom;
    
    ctx.drawImage(
      document.body,
      sourceX, sourceY, sourceSize, sourceSize,
      0, 0, size, size
    );
    
    ctx.restore();
  });
</script>
