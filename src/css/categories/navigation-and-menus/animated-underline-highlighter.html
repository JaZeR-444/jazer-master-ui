Key Features

Smooth Animation

Underline expands from 0 to 100% width using cubic-bezier easing for natural motion.

Gradient Underline

Uses JaZeR's signature gradient (cyan → pink → purple) for vibrant visual appeal.

Glow Effect

Underline features a soft cyan glow shadow for depth and neon aesthetics.

Hover Lift

Links subtly lift upward on hover for enhanced interactivity feedback.

Color Transition

Text color smoothly transitions to cyan on hover to match the underline.

Customizable

Easily adjust thickness, animation speed, and colors via CSS custom properties.
