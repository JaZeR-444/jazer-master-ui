Key Features

Dynamic Sliding

Bar smoothly slides to match the position and width of the hovered or active link.

Neon Glow

Multi-layered box shadow creates authentic neon sign glow beneath navigation items.

Bouncy Animation

Cubic-bezier easing creates playful, bouncy motion that enhances interactivity.

Gradient Bar

Uses JaZeR's signature gradient (cyan → pink → purple) for brand consistency.

Active State

Bar remembers the active link position when not hovering for clear navigation state.

Hover Effects

Links lift slightly and change color on hover for additional visual feedback.

Usage Notes

  • Hover over navigation links to see the neon bar slide beneath them
  • Click a link to set it as active - the bar will return to it after hover
  • Perfect for horizontal navigation bars, tab interfaces, or category selectors
  • Works dynamically with different link widths and spacing
  • Fully responsive with adjusted spacing on mobile devices
  • Uses JavaScript to calculate dynamic bar positioning
