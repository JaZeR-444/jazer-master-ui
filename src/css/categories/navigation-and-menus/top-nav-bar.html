Top Nav Bar

A fixed top navigation bar with logo, links, and action buttons. Features glassmorphism backdrop blur, gradient hover effects, and responsive design. Perfect for application headers and website navigation.

Key Features

Fixed Position

Stays at the top of the viewport while scrolling for easy navigation access.

Glassmorphism

Backdrop blur effect with semi-transparent background for modern aesthetics.

Gradient Accents

Cyberpunk-inspired gradients for logo, hover states, and button highlights.

Active States

Visual indicators showing the current page with animated underlines.

Action Buttons

Primary and secondary buttons for sign-in and sign-up actions.

Responsive

Adapts to mobile screens by hiding links and compacting buttons.

Usage Notes
  • Scroll the page to see the fixed positioning in action
  • Hover over navigation links to see the animated underline
  • Click on links to toggle the active state
  • Resize the window to see responsive behavior
  • Uses JaZeR brand design system for consistent styling
