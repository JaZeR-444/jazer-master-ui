Cyberpunk Navbar

Responsive navigation bar with glowing effects and mobile menu

Features

✓ Fixed position with backdrop blur

✓ Animated gradient border glow

✓ Active state indicators

✓ Hover effects with underline animations

✓ Mobile-responsive hamburger menu

✓ Multiple call-to-action button styles

Usage Tips

The navbar is fixed to the top of the page. Scroll down to see it remain in position.

On mobile devices (resize your browser), the hamburger menu appears and the navigation collapses.

The active link has a cyan glow and underline - click different links to see the effect.
