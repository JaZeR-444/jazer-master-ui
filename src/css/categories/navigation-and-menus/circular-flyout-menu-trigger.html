Key Features

Radial Layout

Menu items positioned in perfect circle around trigger for balanced, accessible design.

Staggered Animation

Sequential reveal of menu items creates smooth, wave-like expansion effect.

Bounce Easing

Cubic-bezier easing adds playful bounce to menu item appearance.

Backdrop Blur

Glassmorphism effect on menu items creates depth and modern aesthetics.

Gradient Trigger

Vibrant gradient button with hover effects draws attention to menu access.

Compact Design

Perfect for floating action buttons or mobile navigation where space is limited.

Usage Notes

  • Click the center button to expand the circular menu
  • Hover over menu items to see scale and glow effects
  • Click again to collapse the menu
  • Perfect for floating action buttons, quick access menus, or mobile navigation
  • Customizable number of menu items (adjusts spacing automatically)
  • Fully accessible with keyboard navigation support
