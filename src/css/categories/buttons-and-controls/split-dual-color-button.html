Split Dual-Color Button

A button with two distinct color halves that swap positions on hover, creating a dynamic color-shifting effect.

CSS Code

.split-dual {
    background: linear-gradient(90deg, var(--jazer-purple) 50%, var(--jazer-pink) 50%);
    border: none;
    color: var(--text-light);
    padding: 1rem 2rem;
    font-family: var(--font-secondary);
    font-weight: 700;
    text-transform: uppercase;
    letter-spacing: 0.05em;
    border-radius: 8px;
    position: relative;
    transition: all 0.3s ease;
    overflow: hidden;
    cursor: pointer;
}

.split-dual::before {
    content: '';
    position: absolute;
    top: 0;
    left: -100%;
    width: 100%;
    height: 100%;
    background: linear-gradient(90deg, var(--jazer-pink) 50%, var(--jazer-purple) 50%);
    transition: left 0.5s ease;
}

.split-dual:hover::before {
    left: 0;
}

.split-dual span {
    position: relative;
    z-index: 1;
}

.split-dual:hover {
    transform: translateY(-2px);
    box-shadow: 0 10px 30px rgba(255, 0, 110, 0.4);
}
