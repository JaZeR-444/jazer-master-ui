Glowing Halo Button

A button with an animated glowing halo effect that pulses around the edges on hover, creating a mystical and engaging visual effect.

CSS Code

.glowing-halo {
    background: var(--bg-card);
    border: 2px solid var(--jazer-cyan);
    color: var(--jazer-cyan);
    padding: 1rem 2rem;
    font-family: var(--font-secondary);
    font-weight: 700;
    text-transform: uppercase;
    letter-spacing: 0.05em;
    border-radius: 50px;
    position: relative;
    transition: all 0.3s ease;
    cursor: pointer;
}

.glowing-halo::before {
    content: '';
    position: absolute;
    top: -10px;
    left: -10px;
    right: -10px;
    bottom: -10px;
    border: 2px solid var(--jazer-cyan);
    border-radius: 50px;
    opacity: 0;
    z-index: -1;
    transition: all 0.3s ease;
}

.glowing-halo:hover::before {
    opacity: 1;
    animation: haloPulse 1.5s ease-in-out infinite;
}

.glowing-halo:hover {
    color: var(--text-light);
    background: var(--jazer-cyan);
    box-shadow: 0 0 30px rgba(0, 242, 234, 0.6);
}

@keyframes haloPulse {
    0%, 100% { transform: scale(1); opacity: 1; }
    50% { transform: scale(1.1); opacity: 0.7; }
}
