Neon Outline Trace Button

A button with animated neon border that traces around the edges on hover, creating a vibrant cyberpunk effect with smooth color transitions.

CSS Code

.neon-outline {
    background: transparent;
    border: 2px solid var(--jazer-cyan);
    color: var(--jazer-cyan);
    padding: 1rem 2rem;
    font-family: var(--font-secondary);
    font-weight: 700;
    text-transform: uppercase;
    letter-spacing: 0.05em;
    border-radius: 8px;
    position: relative;
    overflow: hidden;
    transition: all 0.3s ease;
    cursor: pointer;
}

.neon-outline::before {
    content: '';
    position: absolute;
    top: -2px;
    left: -2px;
    right: -2px;
    bottom: -2px;
    background: linear-gradient(45deg, var(--jazer-cyan), var(--jazer-purple), var(--jazer-pink), var(--jazer-cyan));
    border-radius: 8px;
    opacity: 0;
    z-index: -1;
    transition: opacity 0.3s ease;
    animation: neonGlow 2s linear infinite;
    background-size: 400% 400%;
}

.neon-outline:hover::before {
    opacity: 1;
}

.neon-outline:hover {
    color: var(--text-light);
    transform: translateY(-2px);
    box-shadow: 0 0 20px rgba(0, 242, 234, 0.5);
}

@keyframes neonGlow {
    0%, 100% { background-position: 0% 50%; }
    50% { background-position: 100% 50%; }
}
