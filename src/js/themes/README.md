# Themes

This directory contains JavaScript utilities for managing themes and styling in web applications.

## Files

- `theme-manager.js` - Core theme management functionality
- `theme-components.js` - Theme switching components and utilities
- `theme-utils.js` - Advanced theme utilities and helper functions

## Usage

To use these theme utilities, import the specific module you need:

```javascript
import themeManager from './themes/theme-manager.js';

// Apply a specific theme
themeManager.applyTheme('dark');

// Toggle between light and dark themes
themeManager.toggleTheme();
```
