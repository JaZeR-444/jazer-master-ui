Explore different styles of image grids that adapt gracefully to various screen sizes.

Standard Grid (Fixed Row Height)

Masonry-like Grid (Dynamic Row Height)

Items have varying heights, creating a dynamic, interlocking look. Uses `grid-auto-rows` and image's natural height.

Gapless Grid

A grid where images are directly adjacent to each other without any spacing between them.
