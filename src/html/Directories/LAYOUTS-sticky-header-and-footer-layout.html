Page Content with Sticky Header & Footer

This layout ensures that both the header and the footer remain fixed at the top and bottom of the viewport, respectively, while the main content area scrolls independently.

This pattern is often used for single-page applications or sites where global navigation and branding/legal information need to be persistently visible.
